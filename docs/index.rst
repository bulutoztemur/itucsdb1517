This project aims to store and manipulate the data in order to present it to the users who are curious about one of the most exciting and popular sports that called volleyball.


This platform provides a very user-friendly UI which gives volleyball fans an opportunity to explore the world of this amazing game. Users can reach information about countries, teams, players, statistics, transfers rapidly. Also, application offers a reliable, relational and consistent data to users.
